# Huifu1018 Homebrew Tap

这个仓库提供 `ClawBackup` 的 Homebrew 安装公式。

## Install

先添加 tap：

```bash
brew tap Huifu1018/tap
```

再安装 `ClawBackup`：

```bash
brew install clawbackup
```

也可以一步安装：

```bash
brew install Huifu1018/tap/clawbackup
```

## Upgrade

```bash
brew update
brew upgrade clawbackup
```

## Uninstall

```bash
brew uninstall clawbackup
```

如果不再需要这个 tap：

```bash
brew untap Huifu1018/tap
```

## Formula

当前提供的公式：

- `clawbackup`
