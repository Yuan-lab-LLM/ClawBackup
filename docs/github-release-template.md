# ClawBackup v0.1.0

ClawBackup 的首个公开版本，提供面向 OpenClaw 的本地备份 CLI。

## Highlights

- 支持初始化向导，快速写入源目录和备份目录
- 支持立即备份、历史查看、恢复和删除
- 支持定时任务配置
- 支持 `pipx` 安装
- 支持 Homebrew 公式发布

## Install

### pipx

```bash
pipx install clawbackup
```

### Homebrew

```bash
brew install Huifu1018/tap/clawbackup
```

## Assets

- `clawbackup-0.1.0.tar.gz`

## Notes

- Release tag 使用 `v0.1.0`
- Homebrew 公式仓库建议使用 `Huifu1018/homebrew-tap`
- 如果是手动发布 Homebrew，请同步更新对应版本的 `Formula/clawbackup.rb`
